# `trimprefix` Function

`trimprefix` removes the specified prefix from the start of the given string. If the string does not start with the prefix, the string is returned unchanged.

## Examples

```
> trimprefix("helloworld", "hello")
world
```

```
> trimprefix("helloworld", "cat")
helloworld
```

## Related Functions

* [`trim`](../../language/functions/trim.mdx) removes characters at the start and end of a string.
* [`trimsuffix`](../../language/functions/trimsuffix.mdx) removes a word from the end of a string.
* [`trimspace`](../../language/functions/trimspace.mdx) removes all types of whitespace from
  both the start and the end of a string.
